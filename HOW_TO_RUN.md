# SpeakMate AI — How to Run

## Step 1: Install dependencies
Open terminal in the speakmate-ai folder and run:
```
npm install
```

## Step 2: Run the app
```
npm run dev
```

## Step 3: Open in browser
Go to: http://localhost:3000

Use Chrome or Edge for best voice support.

## Notes
- Your Groq API key is already saved in .env.local
- All data is stored locally in your browser
- Voice features require Chrome or Edge
